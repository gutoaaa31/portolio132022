Sistemas de Numeração

Inverno -T-

Sistemas de numeração são símbolos que usamos para representar quantias, podem ser os símbolos que você bem entender.

Há variados tipos de sistemas de numeração, por exemplo: Decimal, no qual é usado dez símbolos; Binário, no qual são usados dois símbolos; Octal, no qual são usados oito símbolos; Hexadecimal, no qual são usados dezesseis símbolos.

Conversão de base 10 para base 2 e vice e versa

Para fazer a conversão de base 10(decimal) para base 1(binário) dividimos por dois, e o resto das divisões, sendo sempre 0 e 1, é o resuldado de trás pra frente, sendo esse método usado para todas conversões de outras bases pra base 2.

Exemplo:Base 10 pra base 2

464610=1110101102

Exemplo:Base 2 pra base 10

Para converter de base 2(Binário) para base 10(Decimal) você coloca em potência de acordo com a ordem decrescente se for 1, se for 0 não há necessidade. Após isso você calcula as potências e depois soma o resultado de cada uma.

10001110012=56910
